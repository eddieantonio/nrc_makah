Start using the Southern Wakashan keyboard

Type Qʷi·qʷi·diččaq on your phone!

Tutorial

This is a longpress keyboard. This means, to type letters like č, c̓, č̓, press and hold the most similar letter , and the letters will pop-up.

If you want to type č̓, press and hold the c key.
