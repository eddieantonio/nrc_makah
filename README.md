Southern Wakashan keyboard
==============

2021 © National Research Council Canada

Version 1.0

Description
-----------

A longpress keyboard for Southern Wakashan languages:

 - Qʷi·qʷi·diččaq (Makah)
 - nuučaan̓uɫ (Nuu-chah-nulth)
 - diitiidʔaaʔtx̣ (Ditidaht)
